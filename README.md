# Portfolio

This repository contains the source files and content for my academic portfolio website.  


## Contents

The portfolio includes the following sections:

- **About**  
  A brief academic overview and research orientation.

- **Projects**  
  Summaries of computational and physics-related projects, including:
 

- **Skills**  
  Programming languages, scientific libraries, numerical methods, and technical tools used in coursework and research.

- **Blogs / Articles**  
  A collection of technical and academic blog posts covering topics in physics, computation, and related areas.  
  These articles are intended to document learning, clarify concepts, and communicate technical ideas clearly.


- **Coursework**  
  Academic background and relevant coursework.


- **Contact**  
  Professional contact information and external profiles.


##

The website is built using standard web technologies and is hosted via GitHub Pages.
